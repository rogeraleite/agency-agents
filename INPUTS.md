# Inputs - agency-agents

## Purpose
Biblioteca generica de agentes usada como referencia.

## Accepted Inputs
- Brief de adaptacao de agente.
- Pedido de revisao de agente existente.
- Referencia a contrato global quando o output alimentar outro projeto.

## Contracts
- `../../shared_contracts/event.json` para eventos operacionais.
- `../../shared_contracts/article.json` quando gerar documentacao estruturada.

## Input Location
- Templates e briefs locais devem ir para `inputs/`.
