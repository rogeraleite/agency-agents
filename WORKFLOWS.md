# Workflows - agency-agents

## review_agent

1. reviewer le o agente alvo.
2. strategist identifica objetivo e contexto de uso.
3. tester valida clareza de inputs, outputs e restricoes.
4. reviewer entrega recomendacoes em `outputs/`.

## adapt_agent

1. strategist define caso de uso.
2. reviewer seleciona partes reutilizaveis.
3. tester verifica se nao ha acoplamento indevido.
4. salvar adaptacao versionada em `outputs/`.
